# 앱인토스 비게임 검수 가이드

> 출처: https://developers-apps-in-toss.toss.im/checklist/app-nongame.html
> 작성일: 2025-12-30

---

## 사전 필수사항

출시 검토 전에 반드시 완료되어야 하는 항목:
- **앱 정보 검토** 완료
- **사업자 인증** 완료
- **대표관리자 신청 승인** 완료
- **다크패턴 방지 정책** 준수
- **미니앱 브랜딩 가이드** 준수

---

## 1. 시스템 모드

| 항목 | 요구사항 |
|------|----------|
| 다크모드 | ❌ 미지원 (라이트 모드만 제공) |
| 내비게이션 바 | 라이트 모드로 고정 |

---

## 2. 확대/축소 (핀치줌)

- 제스처 핀치줌: **기본 비활성화**
- 모바일 핀치줌 방지 설정:
  ```html
  <meta name="viewport" content="initial-scale=1, maximum-scale=1, user-scalable=no">
  ```
- 예외: 지도 등 필수 기능은 채널톡 문의 후 예외 적용 가능

---

## 3. 내비게이션 바 설정

| 위치 | 항목 | 상태 | 비고 |
|------|------|:----:|------|
| 좌측 | 뒤로가기 버튼 | 선택 | |
| 좌측 | 브랜드 로고+이름 | **필수** | |
| 좌측 | 홈 버튼 | 선택 | |
| 우측 | 제휴사 기능 버튼 | 선택 | 모노 아이콘만 가능 |
| 우측 | 더보기 버튼 | **필수** | |
| 우측 | 닫기 버튼 | **필수** | |

---

## 4. 전반적인 서비스 동작

### 필수 조건
- [ ] 앱 정보의 나이 제한과 실제 내용 일치
- [ ] 스크롤/인터랙션 반응 **2초 이내**
- [ ] 재접속 시 데이터 유지 (작성 중인 내용, 마이페이지 등)
- [ ] 모든 컴포넌트 정상 작동
- [ ] 리스트 정렬·검색·필터링 기능 정상 작동
- [ ] 기능화되지 않은 컴포넌트 미포함
- [ ] 특정 연령층만 이해하는 유행어/밈 사용 금지

---

## 5. 접근성

- [ ] 주요 요소의 명도 대비 충분히 확보
- [ ] 버튼/조작 UI의 터치 영역 확보 (최소 44x44px 권장)
- [ ] 화면 전환·애니메이션이 현기증 유발하지 않도록
- [ ] 스크린 리더 사용 시 의도한 순서대로 콘텐츠 읽힘

---

## 6. 앱 내 기능

- [ ] 모든 제공 기능이 미니앱에서 정상 동작
- [ ] 기능별 앱 스킴 접속 원활
- [ ] 뒤로가기 시 미니앱 메인으로 랜딩

---

## 7. 토스 로그인 (선택)

> RaymondsRisk는 로그인 없이 사용 가능하므로 해당 없음

### 적용 시 필수 조건
- [ ] 토스 로그인만 사용 (자사 로그인/간편 로그인 불가)
- [ ] 콘솔 등록 약관이 정상 연결
- [ ] 약관 확인 후 '동의하고 시작하기' 클릭 시 로그인 진행
- [ ] 닫기 버튼 시 메인으로 복귀
- [ ] 토스앱 설정에서 로그인 끊으면 재접속 시 새로 로그인 요청
- [ ] 인트로 화면을 제공하여 사용자가 서비스 성격과 가치를 이해
- [ ] 닫기 버튼: 인트로에서는 미니앱 닫힘, 중간 플로우에서는 이전 화면으로

---

## 8. 토스페이 (선택)

> RaymondsRisk는 결제 기능 없으므로 해당 없음

### 적용 시 필수 조건
- [ ] 결제 기능 사용 시 토스페이만 가능
- [ ] 주문 금액과 결제창 금액 일치
- [ ] 결제·취소 정상 처리
- [ ] 결제 실패 시 오류 메시지 안내
- [ ] 결제 내역을 미니앱에서 확인 가능

---

## 9. 인앱결제 (선택)

> RaymondsRisk는 인앱결제 없으므로 해당 없음

---

## 10. 프로모션

프로모션 검토 가이드 준수. 다음은 **불가능**:
- ❌ 현금성/환금성 이벤트
- ❌ 아이템·기프티콘·상품권 현금화
- ❌ 사행성·투기성 이벤트

---

## 11. 기능성 메시지 발송 (선택)

> RaymondsRisk는 푸시 알림 없으므로 해당 없음

- 결제 완료, 배송 중, 배송 완료 등 필수 알림만 발송 가능
- 템플릿 형태로 정상 발송

---

## 12. 인앱광고 (선택)

> RaymondsRisk는 광고 없으므로 해당 없음

---

## 13. 앱 사용 권한

- [ ] 콘솔 업로드 후 필요 권한 확인
- [ ] 권한 미동의 시에도 해당 기능 제외한 나머지 정상 작동

---

## 14. 데이터 및 메모리 사용량

- [ ] 과도한 데이터 사용 금지
- [ ] 메모리 누수 없음

---

## 15. 보안

- [ ] 보안 이슈 발생 시 출시 검토 반려 가능
- [ ] HTTPS 통신 필수
- [ ] 민감 정보 암호화
