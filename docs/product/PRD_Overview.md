# Raymontology 제품 개요

> 버전: 1.0 | 최종 수정: 2026-01-23

---

## 1. 제품 정의

### 1.1 제품명
**Raymontology** (Raymond + Ontology)

### 1.2 한 줄 정의
한국 주식시장의 기업-임원-투자자 관계를 온톨로지 기반으로 분석하여 일반 투자자에게 정보 비대칭을 해소하는 금융 리스크 분석 플랫폼

### 1.3 서비스 구성

| 서비스 | 설명 | URL |
|--------|------|-----|
| **RaymondsRisk** | 임원 관계망 + 리스크 신호 분석 | www.konnect-ai.net |
| **RaymondsIndex** | 자본배분효율성 지수 평가 | raymondsindex.konnect-ai.net |
| **앱인토스** | 토스 앱 내 서비스 | intoss://raymondsrisk |

---

## 2. 핵심 가치 제안

### 2.1 정보 비대칭 해소
기관/내부자만 알던 관계망 정보를 일반 투자자에게 제공

### 2.2 관계 네트워크 시각화
임원/대주주/CB 인수인 연결 관계를 직관적으로 시각화

### 2.3 리스크 신호 탐지
전환사채(CB) 남발, 내부자 거래 패턴 등 위험 신호 자동 감지

### 2.4 자본배분효율성 평가
RaymondsIndex를 통한 기업의 자본 활용 효율성 평가

## 4. 온톨로지 모델

### 4.1 핵심 엔티티

```
┌─────────────────────────────────────────────────────────────────┐
│                     RAYMONTOLOGY ONTOLOGY                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   ┌─────────┐         WORKS_AT          ┌─────────────┐         │
│   │ Officer │─────────────────────────▶│   Company   │         │
│   │         │                           │             │         │
│   │ • name  │◀───────────────────────── │ • name      │         │
│   │ • birth │     (임기, 직위)          │ • market    │         │
│   └─────────┘                           └──────┬──────┘         │
│                                                │                 │
│                                          ISSUED│                 │
│                                                ▼                 │
│   ┌────────────┐      SUBSCRIBED      ┌──────────────────┐      │
│   │ Subscriber │─────────────────────▶│ ConvertibleBond  │      │
│   │            │                       │                  │      │
│   │ • name     │                       │ • issue_amount   │      │
│   │ • amount   │                       │ • issue_date     │      │
│   └────────────┘                       └──────────────────┘      │
│                                                                  │
│   ┌────────────┐       SHAREHOLDER_OF                            │
│   │Shareholder │─────────────────────▶ Company                   │
│   │ • name     │                                                 │
│   │ • ratio    │                                                 │
│   └────────────┘                                                 │
│                                                                  │
│   Company ─────── AFFILIATE_OF ──────▶ Company                   │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 관계 유형

| 관계 | 설명 | 속성 |
|------|------|------|
| `WORKS_AT` | 임원 → 기업 | position, term_start, term_end, is_current |
| `ISSUED` | 기업 → CB | - |
| `SUBSCRIBED` | 인수자 → CB | subscription_amount, is_related_party |
| `SHAREHOLDER_OF` | 주주 → 기업 | share_ratio, is_largest |
| `AFFILIATE_OF` | 기업 → 기업 | relationship_type |

## 5. 주요 기능

### 5.1 RaymondsRisk

#### 기업 검색
- 자동완성 검색
- 최근 조회 기업 표시
- 고위험 기업 추천

#### 관계망 그래프
- D3.js 포스 다이렉티드 그래프
- 임원/대주주/CB 인수인 연결 시각화
- 노드 클릭으로 상세 정보 조회
- 확장 노드에서 기업 클릭 시 해당 기업 분석 페이지 이동

#### 리스크 분석
- 5개 리스크 패턴 탐지
  - CB 남발
  - 관련자 CB 인수
  - 적자 지속
  - 고빈도 임원 이동
  - 재무 부실
- 투자등급 표시 (AAA ~ D)
- 리스크 점수 (0-100)

### 5.2 RaymondsIndex

#### 자본배분효율성 평가
- 4대 Sub-Index
  - CEI (자본효율성) 20%
  - RII (재투자강도) 35%
  - CGI (성장건전성) 25%
  - MAI (시장정렬도) 20%
- 9등급 체계 (A++ ~ C)

#### 스크리너
- 등급/점수 필터
- 시장 필터 (KOSPI/KOSDAQ/KONEX)
- 정렬 (점수, 이름, 시가총액)
- 페이징

### 5.3 공통 기능

#### 인증
- 이메일 회원가입/로그인
- 소셜 로그인 (Google, Kakao, Naver)
- 토스 로그인 (앱인토스)

#### 구독
| 이용권 | 가격 | 월 조회 제한 |
|--------|------|-------------|
| Free | 무료 | 5건 |
| Light | 3,000원/월 | 30건 |
| Max | 30,000원/월 | 무제한 |
